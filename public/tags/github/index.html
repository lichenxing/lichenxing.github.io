接着，打开 GitHub，登录到你的账户，按照以下步骤操作：

  1. 点击右上角的个人头像，然后选择 “Settings”。
  2. 在左侧菜单中，选择 “SSH and GPG keys”。
  3. 点击 “New SSH key” 按钮。
  4. 在 “Title” 字段中输入一个描述性名称（例如 “My MacBook”）。
  5. 在 “Key” 字段中粘贴你复制的公钥。
  6. 点击 “Add SSH key” 按钮。
